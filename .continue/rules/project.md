# Project Overview

## General
- Repo: SPOILERJS
- Type: Monorepo con packages/components
- Uses StencilJS, TypeScript, Node

## Folder Structure
- packages/components/src: componentes del UI
- packages/components/dist: distribución compilada
- demo/: demo con GIF y ejemplo
- site/: documentación estática

## Important Files
- package.json: configuración npm
- tsconfig.json: TypeScript
- stencil.config.ts: configuración Stencil
- README.md: documentación principal

## Coding Standards
- TypeScript para todo
- Archivos en camelCase
- Separación de src/components y src/utils
